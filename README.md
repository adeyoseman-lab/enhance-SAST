# SAST Security Framework

Framework untuk memperkaya hasil Semgrep SARIF sehingga lebih informatif di GitHub Code Scanning.

## Fitur (Tahap 1)

- Load database CWE
- Load database OWASP
- Load database CAPEC
- Load database CVSS
- Load mapping rule Semgrep
- Membaca SARIF
- Menulis ulang SARIF
- Upload ke GitHub Code Scanning
